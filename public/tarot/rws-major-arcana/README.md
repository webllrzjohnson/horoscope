# Rider–Waite–Smith Major Arcana artwork

These local tarot card images are sourced from Wikimedia Commons file pages for the Rider–Waite–Smith deck scans (`RWS_Tarot_00_Fool.jpg` through `RWS_Tarot_21_World.jpg`). Wikimedia Commons reports these files as **Public domain**.

The original Rider–Waite–Smith tarot deck artwork is by Pamela Colman Smith, published by Rider & Son in 1909. The scans used here are public-domain files hosted by Wikimedia Commons.

Implementation note: the app serves these from Next.js `public/` static assets, following the official Next.js docs for public assets/local images: https://nextjs.org/docs/app/getting-started/images#local-images
